# Sales Repositioning Notes

## Goal

Reposition the repository from a technical-only health-check starter to a customer-facing technical sales and delivery asset for SEPPmail-related environments.

## What changed

- strengthened Synedat visibility as workshop, implementation and delivery contact
- added customer use cases such as pilot preparation, readiness review and post-change validation
- added explicit official source references for product and partner context
- added image-origin notes for the SEPPmail product visual used on the landing page
- adjusted landing-page wording from purely operational language to business-and-delivery language

## Positioning logic

SEPPmail publicly describes product capabilities on its Secure Email Gateway page and explains on partner and support pages that customers work with accredited partners and certified systems integrators for evaluation, introduction and maintenance. This repository uses that public context to support customer-facing conversations without overstating an unverified public status for Synedat itself.

## Recommended usage

- use the README in initial customer outreach
- use the landing page in workshops and follow-up mails
- use the example outputs for readiness and evidence discussions
- use the workshop kit for pilot or assessment sessions
